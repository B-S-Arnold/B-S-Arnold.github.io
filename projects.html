Instagram Clone Friendstagram

Friendstagram is a clone of Instagram written with a JavaScript and React/Redux frontend and a Python backend. I utilized CSS flexbox and media queries to create a flexible, aesthetic user experience on any screen size including phone and tablet. To create a more user friendly experience, I implemented a drag-and-drop feature for user images and stored them on Amazon Web Services (AWS). I also created a custom search on a persistent nav, so users would always have access to the search bar and navigation icons. This app was a great practice in style, working with new packages and services, and creating an overall good user experience.

Contracting App for Developers deVelp

DeVelp is a group project designed for contracting, reviewing, and listing one's self as a developer. It was written with a JavaScript and React/Redux frontend and a Python backend. The app utilizes geocoding and Google Maps, which can take in any city and state and make a pin for a custom Google Maps page. Google Maps API can only generate a location from longitude and latitude, which is why geocoding was implemented. DeVelp maintains most of the functionality of Yelp, but has a few differences due to the specialized nature of working with developers.

Property Listing Wesbite PopPilgrim

PopPilgrim is clone of AirBnB with a JavaScript frontend and backend. I utilized React/Redux to manage state and manipulate the DOM, and Sequelize was used to create and manage the database. Password protection was implemented using bcrypt and csrf protection to hash, salt, and store hashed passwords in the database giving sitewide security against brute force and csrf attacks. I created full CRUD functionality in a relational database through the use of RESTful routes on the Express backend, providing full control over user-inputted data

Medium Clone Intermedium

Intermedium is a group project and clone of Medium. This was my first full-stack application, and is fully written in JavaScript. I utilized jQuery for frontend event handling and DOM manipulation, and Sequelize to implement the backend. Although jQuery isn't a popular technology, the project was a great practice in finding unique ways to update the DOM and implement full CRUD functionality.
